## 🚀 快速开始

### 环境要求

- Node.js >= 22

### 安装

```bash
# 安装
npm install -g openclaw-cn@latest
# 运行安装向导
openclaw-cn onboard --install-daemon
# 启动网关
openclaw-cn gateway --port 18789 --verbose
